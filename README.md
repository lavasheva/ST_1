## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/lavasheva/ST_1
2. Перейдите в папку проекта: 
    cd расположение-папки
3. Перейдите в папку backend:
    cd backend
4. Запустите сервер:
    node server.js
5. Откройте новый терминал и в нем перейдите в папку frontend:
    cd frontend
6. Установите зависимости:
    npm install
7. Запустите сервер: 
    npm run server
8. В новом терминале перейдите в папку frontend:
    cd frontend
9. Запустите фронтенд:
    npm run dev
10. После запуска откройте браузер и перейдите по адресу:
    http://localhost:5173
